Core Privacy Toggle Plugin

This plugin empowers album owners to manage their galleries directly from their User Control Panel, reducing administrative overhead.

Features (Phase 1)

  • Edit album names and descriptions
  • Toggle album privacy (public ↔ private)
  • Works with Community plugin ownership
  • Fallback support for exclusive contributor detection
  • Progressive enhancement (works without JavaScript)
  • Comprehensive security and accessibility

How to use

Users who own albums (via Community plugin or exclusive contribution) will see a "My Galleries" section in their profile page. No additional configuration is required - the plugin works automatically.
